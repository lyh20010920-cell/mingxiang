# 明祥精密零件有限公司官网

专业精密零件加工制造商官方网站

## 技术栈

- **框架**: Next.js 14 (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **部署**: GitHub Pages

## SEO 功能

- ✅ Meta 标签 (每个页面独立配置)
- ✅ Open Graph 标签
- ✅ Twitter Card 标签
- ✅ sitemap.xml (自动生成)
- ✅ robots.txt (自动生成)
- ✅ Schema.org 结构化数据 (LocalBusiness, Organization, WebSite)

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 部署

推送到 `main` 分支会自动触发 GitHub Actions 部署到 GitHub Pages。

## 页面结构

- `/` - 首页
- `/about` - 关于我们
- `/products` - 产品中心
- `/equipment` - 生产设备
- `/news` - 新闻动态
- `/contact` - 联系我们
